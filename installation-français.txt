Merci d'avoir acheter le "Pack Cote d'Ivoire" pour "Love Dating Script"
==================================================================

Ce pack comprend l'inclusion du Pays Cote d'ivoire sur Love Dating Script
et comprend :

- 13 Région
- 1280 Villes
- 921 Villes géolocaliser

Comment installer le Pack ?
===========================

Il vous suffit de copier le fichier "pack_cote_ivoire.php" sur votre FTP ou est installer
"Love Dating Script" à la racine. Puis de consulter la page "pack_cote_ivoire.php".

Exemple : votre nom de domaine est par exemple http://www.monsite.com, visiter la page http://www.monsite.com/pack_cote_ivoire.php

Une fois le pack installer, vous pouvez supprimer le fichier "pack_cote_ivoire.php" de votre FTP pour éviter
toute tentative de compromision de vos données.
